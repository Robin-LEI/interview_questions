|                | UDP                                                          | TCP          |
| -------------- | ------------------------------------------------------------ | ------------ |
| 可靠性         | 不可靠                                                       | 可靠、安全   |
| 面向对象连接数 | 一对一、一对多、多对一、多对多                               | 只能是一对一 |
| 首部字节大小   | 8字节（首部主要包括端口【源端口、目标端口】，数据报文的校验信息，数据包长度） | 至少20字节   |
|                | 面向无连接                                                   | 面向连接     |
|                | 面向报文                                                     | 面向字节流   |
|                | 视频、直播等                                                 | 文件传输     |
